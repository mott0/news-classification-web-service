Для разработки будут применяться следующие технологии:
1. Frontend часть будет реализована с помощью Angular, фрейморка для разработки веб-приложений, написанного на языке TypeScript.
2. Backend часть будет реализована на языке Java с помощью Spring Framework. Для сборки серверной части приложения используется
фрейморк Maven
