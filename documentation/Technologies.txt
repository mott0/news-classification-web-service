При разработке применялись следующие технологии:
-- backend был разработан на языке программирования Java с помощью Spring Framework;
-- для сборки серверной части приложения использовался фреймворк Maven;
-- frontend был реализован с помощью Angular – фреймворка для разработки веб-приложений, написанного на языке TypeScript;
-- проект и все необходимые документация и данные по нему хранятся в системе контроля версий на сайте сервиса GitHub;
-- для создания обученной модели классификатора использовался оконный инструмент Weka, для классификации пользовательского текста использовалась JAVA-библиотека Weka;
-- База данных новостных статей была создана при помощи DB Browser for SQLite - инструмента для создания, разработки и редактирования файлов баз данных, совместимых с SQLite; для взаимодействия с базой данных применялась SQLite - компактная встраиваемая СУБД;
-- для сборки и развертывания Angular приложения использовалась программная платформа Node.js.
